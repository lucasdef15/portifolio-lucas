# 🚀 Lucas Faria | Portfolio Personalizado

Este é o repositório do meu portfólio profissional. Um projeto desenvolvido com foco em alta performance, animações fluidas e design moderno, destacando minhas habilidades na intersecção entre engenharia de software e tecnologia aplicada.

Disponível para desafios em **Ribeirão Preto - SP** ou remoto.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento front-end do mercado:

- **Core:** React (v18+) & TypeScript
- **Ferramenta de Build:** Vite (Garantindo um ecossistema extremamente rápido)
- **Estilização:** Tailwind CSS (Utilitários de CSS modernos e responsivos)
- **Animações:** Framer Motion & GSAP (Proporcionando transições suaves de alta performance)
- **Ícones:** React Icons
- **Integração de Contato:** Web3Forms API (Envio de e-mails serverless seguro)

---

## ✨ Funcionalidades

- **Interface Futurista (Glassmorphism):** Design limpo e moderno utilizando efeitos de desfoque e gradientes de marca.
- **Animações On-Scroll:** Elementos que surgem de forma fluida conforme o usuário navega pelo site.
- **Formulário de Contato Funcional:** Sistema integrado para recebimento de propostas e mensagens direto na caixa de entrada do e-mail.
- **Download de Currículo:** Download dinâmico do CV atualizado em PDF.
- **Design 100% Responsivo:** Otimizado para dispositivos móveis, tablets e desktops.

## 📦 Como Rodar o Projeto Localmente

1. Clone o repositório:
   ```bash
   git clone [https://github.com/seu-usuario/portfolio-lucas.git](https://github.com/seu-usuario/portfolio-lucas.git)
   ```

````

2. Instale as dependências:
```bash
npm install
````

3. Crie um arquivo .env.local na raiz do projeto e configure a sua chave do Web3Forms:

```plaintext
VITE_WEB3FORMS_KEY=sua_chave_secreta_aqui
```

4. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```
